# Source Boundaries

Raw project materials are not committed to this repository. Their approved, limited roles are:

| Material | Use | Not authoritative for |
|---|---|---|
| NAPS “Cake Layers” workbook | Example of a configurable Region -> Pod -> Territory -> coverage-role hierarchy | Current account ownership, full portfolio coverage, all-GEO schema |
| JPMorgan Chase MVP workbook | Product ideas: concise summary, prioritized signals, metadata, evidence, action | Layout, score weights, labels, account logic, permanent schema |
| Matt / Uzayr check-in notes | Portfolio-to-account workflow; iterative, no-integration starting point | Final requirements, architecture, policy, or universal product rules |

Use these interpretations only. Do not reconstruct, ingest, commit, or expose raw source material without explicit approval.
